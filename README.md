# AI News Digest Generator

An AI-powered news summarizer built using Flask, NewsAPI, and Google Gemini.

## Features

- Search news by keyword
- Filter news by date range
- AI-generated summary using Gemini
- Download summary as PDF
- Modern Bootstrap UI

## Technologies Used

- Python
- Flask
- NewsAPI
- Google Gemini
- ReportLab
- HTML
- CSS
- Bootstrap

## Installation

```bash
git clone https://github.com/yourusername/AI-News-Digest.git
cd AI-News-Digest
pip install -r requirements.txt
python app.py
```

## Environment Variables

Create a `.env` file:

```
NEWS_API_KEY=YOUR_NEWS_API_KEY
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
